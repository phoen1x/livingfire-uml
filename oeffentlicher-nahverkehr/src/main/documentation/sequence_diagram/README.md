### Beschreibung

Folgende Geschäftsfälle sollen mit Hilfe eines Sequence diagram näher beschreiben werden:

* Fahrzeug
 - Fahrzeug fahren
* Ticket
 - Ticket kaufen
 - Ticketkontrolle
* Fahrplanauskunft
* Personenbeförderung
  - Haltestelle durchsagen
  - Haltestelle anfahren
